В ходе выполнения данной отчётной работы была достигнута поставленная цель: разработано программное приложение на языке C\# для расчёта разветвлённой электрической цепи постоянного тока.

В процессе работы были решены все поставленные задачи, а именно:
\begin{enumerate}
    \item Проведён анализ электрической схемы. Рассмотрена цепь со смешанным соединением резисторов. Определены основные параметры, подлежащие расчёту.
    \item Обоснован выбор методов расчёта.
    \item Выполнена проверка баланса мощностей.
    \item Осуществлена проверка законов Кирхгофа.
\end{enumerate}
